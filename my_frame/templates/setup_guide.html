MyFrame Setup Guide

  • Rapsberry Pi (Only tested on Raspberry Pi 4)
  • Monitor with USB Port
  • Mouse and keyboard (I used a Logitech MK270 mouse and keyboard combo)
  • SD card compatible with your Raspberry Pi

You have all your components, and you are ready to start. First things first, we need to setup our Raspberry Pi and install pip. Here are the necessary links for you to follow:

  1. Set up your Raspberry Pi
  2. Install pip on your Raspberry Pi

You have your Raspberry Pi configured and pip is downloaded. Your monitor is connected to a power source, your raspberry pi is connected to a power source, and your mouse and keyboard are connected to the pi!

  1. Register an account on MyFrame. Without an account our API won't pull your images.
  2. Clone the MyFrame repository on Github to your desktop.
    $ git clone https://github.com/TroyLouis/MyFrame.git
  3. Open up a terminal and navigate into your MyFrame folder located on your desktop.
  4. Setup a virtual environment in the MyFrame folder.
    $ python3 -m venv MyFramevenv
  5. Activate the virtual environment.
    $ . MyFramevenv/bin/activate
  6. Install all of the dependencies from requirements.txt with pip
    $ pip install -r requirements.txt
  7. Run the program once the dependencies are installed and a GUI will pop up!
    $ python main.py
  8. Enter your registered email address, select image modifications, and click submit!
